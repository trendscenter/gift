Expanded View Time course

Figure 1: Expanded view of time course

Time course shown in Figure 1 is the expanded view of the time course (See Sorted Components Figure). The expanded view time course figure has menus like "Options" and "Utilities". The explanation for these two menus is given below:

"Options" Menu : "Options" menu contains the following sub menus:

  • "Time course Options"
  • "Adjust ICA"

"Utilities" Menu: "Utilities" menu contains the following sub menus:

  • "Fast Fourier Transform" - Computes the power spectrum of the ICA time course.
  • "Split Time courses" - Shows the split of the time courses.
  • "Event Average" - Calculates the event average of the ICA time course by selecting the onsets of the specified reference function.

Note: Figures of the utilities will be shown in a new figure using the information of the time course plotted in the expanded view plot.
