Display GUI

When you click Display button in the toolbox and had selected the parameter file which you gave during the analysis, the following figure will be displayed. The parameters in the figure are explained below:

Figure 1: Display GUI

  • 'Select display method' - There are two display methods like component and subject. Each display method is explained below:
    • Component display method displays all components of a viewing set. You can also select the components of interest in the component list box.
    • Subject display method displays a component of all viewing sets.
  • 'Viewing Set' - You can look at mean of all data sets, mean of one session, individual data set components, etc.
  • 'Component No:' - Component numbers you want to look at. For subject display method only one component can be selected whereas for component display method more than one component can be selected.
  • "Display Defaults" menu - Display defaults are plotted in a figure which will open when you click "Display Defaults" menu.
  • "Options" menu contains items like 'Select EEG channel locs file' and 'Select time scale'. Each item is explained below:
    • When you click 'Select EEG channel locs file', a figure will open to select the channel locations file for doing topography plots for components. The default color map for topography plot is in 'icatb_defaults.m' file with the variable name EEG_TOPOPLOT_COLORMAP.
    • 'Select time scale' - This utility is optional. EEG signal time axis in milliseconds can be loaded from an ASCII or a SET file.
