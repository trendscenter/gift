SPM File Selection

When you select "Design Matrix" option in "Display GUI Options" menu (Display GUI). After the "Design Matrix" option is selected, a figure window will open with two possible options. The options are shown below:

Figure 1: Method for selecting the regressors

There are three options for selecting the regressors. In the figure window only two are shown because the example data-set has three subjects and one session. The options are explained as follows:
Same set of regressors for all data sets
Same set of regressors are selected for all data-sets.
Different regressors for sessions
Session specific regressors can be selected.
Different regressors for subjects and sessions
Different regressors for each individual data-set can be possible. Selecting design matrix and regressors can be tedious using GUI. Option is provided in batch script to enter the design matrix for each subject. Regressors can be entered through a text file (See METHOD_ENTERING_REGRESSORS variable in defaults).
