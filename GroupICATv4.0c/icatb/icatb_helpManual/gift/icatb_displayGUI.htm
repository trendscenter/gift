Display GUI

When you click Display GUI  button in the toolbox and had selected the parameter file which you gave during the analysis, the following figure will be displayed.

Figure 1: Display GUI

Display GUI includes Subject Component Explorer in addition to the three visualization methods. By default, Component Explorer is selected after you selected the parameter file. The selected visualization method will be highlighted in colored text. The functions of each control are explained below:

Sort Components
Components will be sorted spatially or temporally. This control will be enabled only for Component Explorer visualization method.
Viewing Set
This is the component viewing set to look at like subject 1 session 1, mean for session 1, etc and will be disabled for Subject Explorer visualization method.
Component number
Component number/numbers to look at. This will be disabled for Component Explorer visualization method.
Display Defaults menu.
Some of the controls are shielded from the user. These will be displayed after you click Display Defaults menu.
Display GUI Options Menu
Display GUI Options menu contains sub-menu Design Matrix
  • Design Matrix - Option is provided to select the design matrix, which will be used to sort components temporally.
Component
Displays all components of a viewing set.
Subject
Displays a specific component from all viewing sets.
Orthogonal
Displays a component with the three views (axial, saggital and coronal).
Composite
Multiple components of a particular subject's session are overlaid.
Load Anatomical
Load Anatomical button is used to select anatomical image. Component images will be overlaid on this anatomical image. By default, first image of functional data will be used as an anatomical image.
Display
Display button is used to display the components of different visualization methods.
