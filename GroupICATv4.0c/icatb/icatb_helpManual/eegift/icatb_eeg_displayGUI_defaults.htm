Display Defaults

The display defaults are as follows:

Image Values
There are four options like 'Positive and Negative', 'Positive', 'Absolute' and 'Negative'. The default value is 'Positive and Negative'.
Convert To Z-scores
You have the option to convert EEG signal of all trials to Z-scores. For now select 'Yes'. If you are looking at t-statistic images, do not convert to Z-scores.
Threshold Value
Z threshold applied on image. The default value is '1.0'.
Components Per Figure
Options are '1', '4' and '9'. The default value is '4.0'.
